Best Practices
  • Start Conservative: Begin with higher limits and adjust based on monitoring
  • Monitor First: Use "Pass" action initially to observe traffic patterns
  • Per-IP Limits: Usually lower than global limits to prevent single-source attacks
  • Burst Allowance: 10-20% works well for most applications
  • Window Size: Shorter windows (1-5s) for real-time protection
